## Installation

To run the project locally, follow these steps:

1. Clone the repository to your local machine:

```bash
git clone https://github.com/dilanrojas/recipe-page.git
```

2. Install dependencies using NPM

```bash
npm install
```

2. Initialize the project

```bash
npm run dev
```
